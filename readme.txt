Nasıl çalıştıracaksınız.

1. "https://www.apachefriends.org/tr/index.html" bu siteden xampp'i indirin.
2. xampp'i çalıştırın ve Apache, Mysql yazan yerlerin yanındaki butonlardan start butonuna basın ve çalıştırmaya başlayın.
3. Ardından Mysql yazısının yanında Admin Butonu olacak oraya basın ve PhpMyAdmin Sayfasını açın.
4. Sol tarafta bulunan linklerden +Yeni linkine tıklayın ve Gelen sayfada istediğiniz veritabanı adına Website yazıp oluştur deyin.
5.Önünüze gelecek sayfada Tablo adı ve sütun sayısı olacaktır. Tablo adını " users " yapın ("" işaretleri olmadan). Sütun sayısını ise 3 yapın ve oluştur deyin.
6.Önünüze Ad, Tür gibi değerler olan bir tablo gelecek. Burada ad bölümüne sırasıyla Name, Email ve Password yazın. Hepsinin türünü text seçin. Ve aşağıdan kaydet deyin.
7.Visual Studio Code ile verdiğim dosyayı açın. 

8.Yukarıdan terminal'e tıklayıp new terminal deyin.
Aşağıda gelen terminale sırasıyla bunları yazın.
pip install Flask
pip install wtforms
pip install email_validator
pip install passlib

9.Dosyayı sağ yukarıdan çalıştırın.
10.http://127.0.0.1:5000 adresine gidin ve çalışıyor mu diye test edin.
